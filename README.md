## 專案介紹

這是一個心理測驗網站，通過回答一系列問題來測試您的肥胖程度。測驗基於國家肥胖研究院的最新研究，以輕鬆幽默的方式分析您的飲食心理模式。

### 主要功能

- 10題精心設計的肥胖心理測驗
- 分析四種肥胖特質：食物執念、運動迴避、自我欺騙、零食創意
- 根據答案給出您的肥胖類型
- 支援分享和匯出測驗結果圖片
